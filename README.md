## Overview

Windsurf Professional Suite is a complete AI development platform with team features, model flexibility, and enterprise controls for professional AI-assisted development.

Team knowledge sharing builds shared understanding of organizational codebases, conventions, and patterns that improve AI suggestions across the team.

Model selection supports multiple AI providers and local models for organizations with specific model preferences or data sovereignty requirements.

VS Code extension compatibility preserves access to the complete VS Code extension ecosystem.

Windsurf Professional Suite is the AI development platform for teams that need deep codebase-aware AI assistance with the team features and enterprise controls that professional software development requires.

## Performance & Integration

Windsurf is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Windsurf processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows

## System Requirements

- macOS 12 Monterey or later
- Apple Silicon or Intel — Universal Binary
- Works with macOS Ventura, Sonoma and Sequoia
